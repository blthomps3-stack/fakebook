**Link**: https://blthomps3-stack.github.io/fakebook//br

**Author**: Blake Thompson

**Purpose**: Simulate the posting system of a social network

**Features**  
Pop-up modal in the header that fetches and displays user info (user info is hardcoded, modal is not)  
Postbox that allows you to attach photos and write text for posts (post will be discarded if there's no caption or image)  
Timeline that displays submitted posts  
Posts are not stored, if you refresh the page the timeline will be cleared
